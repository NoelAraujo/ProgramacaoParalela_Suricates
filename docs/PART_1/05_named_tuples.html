5  Mantimentos

Depois de entregar rapidamente resultados bem-sucedidos, Rocco atribuiu uma nova tarefa a Daisy. Ela já havia usado códigos binários para enviar um sinal de espionagem, que foi eficaz para esse propósito. Contudo, para a tarefa de controle logístico, estes códigos não eram adequados. O departamento de suplementos precisava de um relatório detalhado sobre quais itens carregar no caminhão, especificamente um menu dos itens transportados.

Daisy precisava criar um arquivo de texto com este menu e salvá-lo em uma pasta designada. A mensagem precisava ser clara, sem a necessidade de quaisquer elementos gráficos.

A equipe da cozinha teve que fornecer informações sobre as opções de menu disponíveis. Dadas as opções limitadas do deserto, o cardápio precisava ser simples e consistente, listando todas as opções disponíveis para o mês inteiro, sem preferências ou ordem específica.

Como os fóruns da internet eram inacessíveis na rede militar, Daisy procurou a ajuda de Eugênio. Ele estava prestes a sair para seus treinos e a encaminhou para Martin, um especialista em programação paralela que foi convidado por Rocco devido à sua experiência anterior com um grupo de hackers. Martin era responsável pelo radar (que ainda não estava operacional), mas estava disponível para auxiliar em questões técnicas. Ele também era próximo de Daisy e Eugenio.

Martin não poderia escrever os códigos para Daisy, pois isso iria contra as ordens de seu superior. No entanto, ele sugeriu que ela usasse um código com uma variável diferente – um objeto que pudesse armazenar tanto o dia da semana quanto os itens de menu correspondentes.

Após alguns momentos de ansiedade e frustração, Daisy conseguiu encontrar uma solução para o problema.

"""
    gerar_cardapio!(mes, menu)

Criar tuplas nomeadas para ser enviado nos dias indicados no `mes`, baseado no `menu` disponivel
# Argumentos
- `mes`: matriz booleana, que indica quais dias da semana serão enviados mantimentos
- `menu`: array de strings com os possíveis insetos a ser enviados para as tropas
"""
function gerar_cardapio(mes, menu)
    dias_da_semana = ["Segunda", "Terça", "Quarta", "Quinta", "Sexta", "Sábado", "Domingo"]
    T = eltype(mes)
    sim = one(T)

    cardapio = map(eachrow(mes)) do semana
        [ (dia = dias_da_semana[i], inseto = rand(menu)) for i in eachindex(semana) if semana[i] == sim ]
    end

    return cardapio
end

using Random; Random.seed!(2024)

menu = ["Besouro", "Aranha", "Escorpião", "Gafanhoto"]
proximas_entregas = rand(Bool, 4, 7) # apenas para testes pode ser aleatorio
cardapio = gerar_cardapio(proximas_entregas, menu)

for (i, semana) in enumerate(cardapio)
    println("Na semana $(i) deverá ser enviado")
    for historico in semana
        println("  - $(historico.dia): $(historico.inseto)")
    end
end

5.1 Tuplas Nomeadas

Tupla Nomeada são variáveis que são armazenadas juntas, e você não poderá alterar seus valores depois.

Sintaxe: Named Tuple

Você cria uma tupla usando parênteses, e o respectivo nome da variável. Não se esqueça da vírgula. Mesmo com apenas 1 elemento.

a = 1
b = 2
c = (variavel_a = a, variavel_b = b)
@show c.variavel_a, c.variavel_b

# ou apenas 1 elemento usa a virgula
c = (variavel_a = a, );
(c.variavel_a, c.variavel_b) = (1, 2)

5.2 Valores Aleatórios

Sintaxe: rand

Por padrão a função rand() gera valores aleatórios entre 0 e 1, mas você pode fornecer uma coleção (como um array de Strings) e a função rand irá sortear valores aleatórios baseados no conteúdo da coleção.

menu = ["aranha", "besouro", "escorpião", "gafanhoto"]
@show rand(menu);
rand(menu) = "besouro"

Além disso, você pode construir vetores ou matrizes

display(rand(menu, 3, 2)) ;
3×2 Matrix{String}:
 "gafanhoto"  "aranha"
 "besouro"    "gafanhoto"
 "besouro"    "besouro"

5.3 Compreensão de Listas

É uma maneira compacta de criar dados, já presente em Python por exemplo, que assim como a função map pode deixar a legibilidade do código curta e/ou clara.

Sintaxe: List Comprehension vs [] + push!

Daisy esta aprendendo as novas regras de segurança, e não é permitido entrar com certos itens dentro da zona militar. Usando uma compressão de lista, vamos filtrar os itens válidos que Daisy levou em sua mochila no seu primeiro dia de trabalho.

itens_mochila = ["perfume", "chocolate", "camiseta", "livro", "chaveiro", "tablet"]
itens_validos = ["chocolate", "livro", "camiseta" ]

itens_permitidos = [item for item in itens_mochila if item in itens_validos]
3-element Vector{String}:
 "chocolate"
 "camiseta"
 "livro"

Uma maneira equivalente de escrever o mesmo código é usando Array vazio e com o comando push!, repare que existe um ! no nome, ou seja, ela altera a variável de entrada.

itens_permitidos = []
for item in itens_mochila 
    if item in itens_validos
        push!(itens_permitidos, item)
    end
end
@show itens_permitidos
itens_permitidos = Any["chocolate", "camiseta", "livro"]
3-element Vector{Any}:
 "chocolate"
 "camiseta"
 "livro"
Exercício 5.3

Daisy já esta pensando em como gastar seu primeiro salário, ele irá comprar produtos da Shopee. Contudo, ela tem que pagar uma taxa de 30% ao valor dos itens importados. Primeiro o valor de cada item é convertido para Pula botsuano, e todos itens abaixo de 500 pulas são livres de imposto. Como Daisy esta fazendo compras de produtos da Namibia, a conversão de Dolar namibiano para Pula botsuano é de 0.7322991.

Converta o código abaixo que utiliza a função mapreduce em uma Lista de Compreensão.

Obs: mapreduce é parecido ao map, porém realiza uma operação extra, por exemplo adição, em todos os itens que são calculados.

valor_dolar_namibia = [30, 750, 1547]
fator_conversao = 0.7322991

valor_liquido = mapreduce(+, valor_dolar_namibia) do item
    valor_pula = item*fator_conversao
    if  valor_pula >= 500
        valor_pula
    else
        0.0
    end
end

valor_bruto = 1.3*valor_liquido
2186.71834251
Solução 5.3 
valor_dolar_namibia = [30, 750, 1547]
fator_conversao = 0.7322991

valores_convertidos = valor_dolar_namibia.*fator_conversao
valor_liquido = sum([ item for item in valores_convertidos if item > 500])

valor_bruto = 1.3*valor_liquido
2186.71834251
