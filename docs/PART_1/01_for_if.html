1  Sair de Casa

Certa vez, nas planícies gramadas do deserto de Kalahari, uma pequena suricate chamada Daisy vivia com sua família. Daisy era uma suricata curiosa e aventureira, sempre desejosa de explorar a vastidão do mundo além de sua toca. No entanto, todos os dias, antes de se aventurar, Daisy pedia a permissão de seus pais para ir caçar.

No início, Daisy se aproximava dos pais e perguntava: “Posso ir caçar hoje?”. Eles acenavam com a cabeça em sinal de aprovação e ela saía correndo, sempre voltando com a barriga cheia de insetos e larvas deliciosos. Mas, com o passar do tempo, o fascínio de Daisy pela tecnologia cresceu e ela começou a pensar em uma maneira mais conveniente de pedir permissão.

Determinada a encontrar uma solução, Daisy embarcou em uma missão para criar seu próprio programa de celular. Usando suas patas ágeis e um pouco de desenvoltura, ela chegou a um resultado convincente e apresentou aos seus pais, com a empolgação borbulhando em seus olhos. Seus pais trocaram olhares confusos, mas decidiram se entregar ao espírito inventivo de Daisy. Eles acenaram com a cabeça e permitiram que ela fosse caçar respondendo ao seu aplicativo.

Daquele dia em diante, o app de Daisy tornou-se parte integrante de sua rotina diária. Todas as manhãs, ela apertava os botões de seu aparelho e já descobria de forma automatizada se seus pais lhe conderam permissão ou não para uma ótima caçada.

Abaixo você tem um código em Julia que é uma simplificação do algoritmo utilizado por Daisy. O código usa um loop para percorrer cada dia da semana e um gerador de números aleatórios para determinar se Daisy tem ou não permissão para sair de casa naquele dia.

dias_da_semana = ["Segunda","Terça", "Quarta", 
    "Quinta", "Sexta","Sábado", "Domingo"]

for dia in dias_da_semana
    # Daisy consegue permissão para sair
    # de casa com probabilidade 50%
    permissão = rand() > 0.5
    if permissão == true
        println("Daisy PODE CAÇAR na $(dia)")
    else
        println("Daisy FICA EM CASA na $(dia)")
    end
end

1.1 for

Dado uma coleção, por exemplo um Array, você pode acessar cada elemento da seguinte forma

for 
coleção = ["a", "b", "1", "2"]
for item in coleção
    println(item)
end
a
b
1
2
Exercício 1.1

Preencha os espaços em branco a seguir para que meses_do_ano armazene Strings com todos os meses do ano, e os exiba na tela.

meses_do_ano = ______________
for _____ in meses_do_ano
    println(_____)
end
Solução 1.1 
meses_do_ano = ["Janeiro", "Fevereiro", "Março", "Abril", 
"Maio", "Junho", "Julho", "Agosto", "Setembro", 
"Outubro", "Novembro", "Dezembro"]
for mês in meses_do_ano
    println(mês)
end

Outra alternativa é ter uma variável com o índice de cada elemento

eachindex 
coleção = ["a", "b", "1", "2"]
for idx in eachindex(coleção)
    valor = coleção[idx]
    println("indice $(idx) tem valor $(valor)")
end
indice 1 tem valor a
indice 2 tem valor b
indice 3 tem valor 1
indice 4 tem valor 2
Exercício 1.2

Preencha os espaços em branco no código fornecido para obter a saída desejada.

coleção = ["a", "b", "1", "2"]
for idx in eachindex(coleção)
    valor = coleção[idx]
    println("______ $(______) tem valor $(______)")
end

Saída Desejada:

índice 1 tem valor a
índice 2 tem valor b
índice 3 tem valor 1
índice 4 tem valor 2
Solução 1.2 
coleção = ["a", "b", "1", "2"]
for idx in eachindex(coleção)
    valor = coleção[idx]
    println("indice $(idx) tem valor $(valor)")
end

Contudo, a maneira recomendada para produzir indices e valores ao mesmo tempo é com enumerate

enumerate 
coleção = ["a", "b", "1", "2"]
for (idx, valor) in enumerate(coleção)
    println("indice $(idx) tem valor $(valor)")
end

1.2 if

Condicionais em Julia usam o comando if

ìf 
condição_1 = true
condição_2 = true
if condição_1 == true
    println("entrou condição 1")
elseif condição_2 == true
    println("entrou condição 2")
else
    println("entrou no else")
end
entrou condição 1

Se a condição_1 já é uma variável booleana (true/false), a comparação pode ser omitidida

ìf simplificado 
condição_1 = false
condição_2 = true
if condição_1
    println("entrou condição 1")
elseif condição_2
    println("entrou condição 2")
else
    println("entrou no else")
end
entrou condição 2
