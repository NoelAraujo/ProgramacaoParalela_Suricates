3  O Convite

Semanas de treinamento se passam, o amanhecer traz novos dias na colônia e, de surpresa, o líder Rocco chega ao centro de treinamento de Eugenio. Antes de se tornar líder militar, Rocco se formou em ciência da computação e teve carreira no infame grupo de hackers Anonymous Desert. Ele ansiava por aventura que nunca poderia encontrar sentado atrás de uma mesa, então abandonou sua carreira na tecnologia e agora serve a família real como líder militar, sendo uma de suas funções cuidar de todo o equipamentos.

Rocco acordara a todos no início da manhã para um anúncio. O exército adquiriu um poderoso computador que será usado para controlar os radares e finalmente rastrear aqueles pássaros irritantes. Eles agora precisam de voluntários dos setores militar e civil para trabalhar neste projeto.

Claro, o representante da equipe de guerreiros conhecia o candidato perfeito para o trabalho: Eugenio, o suricate falante que provavelmente até suas pulgas tomavam creatina e whey protein.

Finalmente, Eugenio se livra da tarefa de decorar como usar canhões antiaéreos que outros suricatos roubaram de um acampamento militar humano. Fazia um certo tempo que aquele dispositivo era apenas brinquedo, pois sem um radar funcionando, não era possível mirar e atirar naquelas galinhas voadoras durante tempestades de areia.

Os guerreiros em treinamento não queriam Eugenio por perto, e ele ao mesmo tempo, estava ansioso para escapar do seu destino de estar na linha de frente. Agora, a oportunidade perfeita surgiu: um trabalho com computadores que o colocaria longe do perigo.

Eugenio aceita feliz a oferta para o novo emprego, mas também tem um pedido: ele quer que sua amiga civil, Daisy, também se junte à equipe. Rocco fica surpreso, mas sabe que precisa de mais do que apenas um rabo que saiba programar. Depois de pensar um pouco, ele diz a Eugenio que para um suricato civil ser considerado para o cargo, ele deve enviar um código de amostra mostrando sua experiência na linguagem Julia resolvendo um desafio.

O desafio é escrever um código que possa se comunicar sem usar palavras ou voz.

Naquela noite, Eugenio corre para a casa de Daisy e conta a ela sobre sua oportunidade. No entanto, há um porém - de acordo com os termos de Rocco, Eugenio não pode ajudá-la. Daisy deve completar o desafio sozinha. Para a surpresa de Eugenio, Daisy estava mais do que preparada para o desafio. Daisy respondeu o desafio com apenas 3 linhas de código. Sua resposta foi um código especial que só Rocco reconheceria pelo seu passado.

map(["ser anonymous não é invadir códigos",  "ser anonymous é ser o código"]) do text
    replace(text, "anonymous" => "?")
end
2-element Vector{String}:
 "ser ? não é invadir códigos"
 "ser ? é ser o código"

3.1 Decifrando a mensagem

Utilizamos previamente a função map porém sem muitos detalhes.

Agora vamos compreendê-lá, pois será utilizada recorrenmente no futuro próximo.

map

A função map aplica uma função, por exemplo, função chamada foo, a cada elemento em uma coleção.

O retorno dessa função também é uma coleção do mesmo tipo.

foo(x) = replace(x, "A" => "B")
respostas = map(foo, coleção)

O código equivalente a esse exemplo seria

foo(x) = replace(x, "A" => "B")
respostas = []
for item in coleção
    push!(respostas, foo(item))
end

Porém, outra maneira de escrever o mesmo código é utilizando uma Função Anônima

map( x->replace(x, "A" => "B"), coleção)
Exercício 3.1

O código a seguir obtêm os dias da semana que Daisy pode sair pra caçar com uma função anônima dentro da função filter.

Escreva um novo código, com for e if, afim de realizar a mesma operação que o filter

filter(x->!isequal(x, "Em Casa"), ["Em Casa", "Em Casa", "Ter"])
1-element Vector{String}:
 "Ter"
Solução 3.1 
dias_caçada = String[]
for dia in ["Em Casa", "Em Casa", "Ter"]
    if !isequal(dia, "Em Casa") # if dia ≠ "Em Casa"
        push!(dias_caçada, dia)
    end
end

Nesse exemplo, a função temporária possui apenas 1 comando (replace). Porém, códigos mais interessantes possuem vários comandos

O que fazer se sua função anônima possui muitas linhas?

Nesse cenário devemos usar a sintaxe do bloco do.

do

Qualquer função cujo primeiro argumento é uma função (como é o caso da função map) pode ser reescrita com a sintaxe do em uma função anônima. Vejamos o exemplo.

map(coleção) do x
    replace(x, "A" => "B")
end

Vale ressaltar que em códigos com apenas 1 comando, essa sintaxe pode ser muito complicada ou descessária, contudo, para funções com mais instruções, essa sintaxe se torna muito conveniente. Vejamos outro exemplo, primeiro com o uso do bloco do.

coleção = ["A", "a", "B", "b"]
map(coleção) do x
    y = replace(x, "A" => "B")
    z = replace(y, "a" => "b")
    z
end
4-element Vector{String}:
 "B"
 "b"
 "B"
 "b"

A versão alternativa, sem bloco do, necessaria criar uma função anônima com a sintaxe ->, além de ser necessário a adição de begin e end

map( x-> begin 
    y = replace(x, "A" => "B"); 
    z = replace(y, "a" => "b"); 
    z end, coleção)
Exercício 3.2

Daisy recebeu um novo castigo. Ela pode caçar todas as Qua e Qui, mas nos outros dias, a suricata ficará em casa.

Escreva um código que reflita a nova situação de Daisy com o comando map.

Obs: não existe mais chances envolvidas, Daisy só poderá caçar em tais dias.

Solução 3.2 
dias = ["Seg", "Ter", "Qua", 
        "Qui", "Sex", "Sab", 
        "Dom"]
dias_caçada = map(dias) do dia
    if dia in ["Qua", "Qui"]
        "pode caçar"
    else
        "fica em casa"
    end
end

3.2 Usando o map

Daisy decidiu usar a função map em seus códigos faz um tempo. Por exemplo, a sua implementação da função calendario_semanal que Eugenio conhecia, ficou diferente.

function calendario_semanal(dias)

    agenda = map(dias) do dia
        if rand() > 0.5
            "sim"
        else
            "não"
        end
    end

    dias_caçada = findall( agenda .== "sim" )
    nomes_dias_caçada = dias_da_semana[dias_caçada]
    return nomes_dias_caçada
end
Exercício 3.3

Altere a função anônima dentro do map, de tal forma que você possa substituir o comando findall pelo comando filter.

Solução 3.3 
function calendario_semanal(dias)
    agenda = map(dias) do dia
        if rand() > 0.5
            dia
        else
            "não"
        end
    end

    nomes_dias_caçada = filter(x->!isequal(x, "não"), agenda)
    return nomes_dias_caçada
end
